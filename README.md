# Smart Traffic Analysis System

A desktop application for monitoring and analyzing traffic data, detecting violations, and managing vehicle information.

## Features
- Graphical User Interface (GUI) using Tkinter
- SQLite database for storing vehicle and violation data
- Speed violation detection
- Vehicle information tracking (speed, color, number plate)
- Fine management system

## Setup Instructions
1. Install Python 3.x
2. Install required packages:
   ```
   pip install -r requirements.txt
   ```
3. Run the application:
   ```
   python main.py
   ```

## Project Structure
- `main.py`: Main application entry point
- `database.py`: Database operations and schema
- `gui.py`: GUI components and interface
- `requirements.txt`: Project dependencies
